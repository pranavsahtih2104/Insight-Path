# 🚀 InsightPath – AI Decision Exploration Platform

## 📌 Overview

**InsightPath** is an AI-powered decision support system designed to help users analyze real-life scenarios and make informed decisions. The platform leverages structured reasoning, intelligent insights, and visualization to guide users through complex decision-making processes.

It transforms ambiguous problems into **clear, data-driven choices** using AI-generated recommendations and interactive exploration.

---

## 🎯 Objectives

* Provide structured decision-making assistance
* Generate intelligent insights using AI
* Help users evaluate multiple options effectively
* Visualize outcomes for better understanding
* Improve clarity in complex real-world scenarios

---

## 🧠 Key Features

### 1. Scenario Input

* Users describe real-life problems or situations
* Accepts flexible, natural language input

### 2. Decision Generation

* AI suggests multiple possible decisions
* Covers diverse approaches and alternatives

### 3. Insight Generation

* Provides reasoning, pros & cons, and impact analysis
* Helps users understand consequences

### 4. Decision Selection

* Users can compare and select the best option
* Supports rational evaluation

### 5. Visualization

* Graphical representation of decisions and outcomes
* Improves clarity and interpretability

### 6. Data Management

* Stores user scenarios and decisions
* Enables tracking and analysis

## 🚀 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/insightpath.git
cd insightpath
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env` file and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection
API_KEY=your_ai_api_key
```

### 4. Run the application

```bash
npm start
```

### 5. Open in browser

```
http://localhost:5000
```
